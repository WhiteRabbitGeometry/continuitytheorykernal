# Continuity Lens Use Cases

Continuity Lens can be applied wherever persistence through change is relevant.

## Literature

Questions:

* What continuity is under pressure?
* What is inherited?
* What is forgotten?
* What is restored?

Examples:

* The Odyssey
* Waiting for Godot
* Moby-Dick
* Hamlet
