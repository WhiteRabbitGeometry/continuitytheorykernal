# Example: Personal Identity

## Subject

A human individual across time.

## Continuity Analysis

### Subject Continuity

The persistence of self through physical, psychological, and experiential change.

### Inherited Structures

- Memory
- Personality tendencies
- Embodied habits
- Language
- Narrative identity

### Integrity Strengths

- Self-reflection
- Memory consolidation
- Social recognition

### Continuity Debts

- Trauma
- False memories
- Cognitive bias

### Failure Modes

- Traceability Failure
- Fragmentation
- Severance
- Recursive Closure

### Key Question

What must remain continuous for the individual to remain recognizable as the same self?

### Thesis

Identity emerges from continuity rather than causing it.
